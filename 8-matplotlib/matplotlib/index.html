# Matplotlib untuk Visualisasi Data

# Apa itu Matplotlib?

Matplotlib adalah library visualisasi data yang paling populer di Python. Library ini memungkinkan pembuatan berbagai jenis grafik dan plot dengan mudah dan fleksibel. Dengan Matplotlib, dapat dibuat grafik seperti garis, batang, scatter, histogram, pie, dan banyak lagi, baik untuk keperluan analisis data maupun presentasi.

# Instalasi Matplotlib

Jika Matplotlib belum terpasang di Python, instal dengan menggunakan pip:

pip install matplotlib

# Import Matplotlib

Untuk mulai menggunakan Matplotlib, perlu mengimpornya terlebih dahulu. Biasanya, Matplotlib diimport dengan alias plt untuk memudahkan penulisan kode.

import matplotlib.pyplot as plt

# Struktur Dasar Plot di Matplotlib

Struktur dasar pembuatan plot di Matplotlib melibatkan beberapa langkah utama:

  1. Membuat Figure dan Axes: Figure adalah kanvas keseluruhan, sementara Axes adalah area tempat plot digambar.
  2. Membuat Plot: Menambahkan grafik ke Axes.
  3. Menampilkan Plot: Menampilkan grafik yang telah dibuat.

# Contoh Sederhana 

import matplotlib.pyplot as plt

# Data yang akan dipetakan
x = [1, 2, 3, 4, 5]
y = [2, 3, 5, 7, 11]

# Membuat plot garis
plt.plot(x, y)

# Menambahkan judul dan label sumbu
plt.title("Plot Garis Sederhana")
plt.xlabel("Nilai X")
plt.ylabel("Nilai Y")

# Menampilkan plot
plt.show()

Penjelasan:

  • plt.plot(x, y): Membuat plot garis dengan data x dan y.
  • plt.title(), plt.xlabel(), plt.ylabel(): Menambahkan judul plot dan label pada sumbu X dan Y.
  • plt.show(): Menampilkan plot yang telah dibuat.

Output:

# Jenis-Jenis Plot di Matplotlib

Matplotlib mendukung berbagai jenis plot yang dapat digunakan sesuai kebutuhan analisis data.

# 1. Plot Garis (Line Plot)

Plot garis digunakan untuk menampilkan tren data seiring waktu atau urutan tertentu.

import matplotlib.pyplot as plt

# Data
x = [0, 1, 2, 3, 4, 5]
y = [0, 1, 4, 9, 16, 25]

# Membuat plot garis
plt.plot(x, y, marker='o', linestyle='-', color='b')

# Menambahkan judul dan label
plt.title("Plot Garis: y = x²")
plt.xlabel("X")
plt.ylabel("Y")

# Menampilkan plot
plt.show()

Penjelasan:

  • marker='o': Menambahkan marker berbentuk lingkaran pada setiap titik data.
  • linestyle='-': Mengatur jenis garis sebagai garis lurus.
  • color='b': Mengatur warna garis menjadi biru.

Output:

# 4. Histogram

Histogram digunakan untuk menampilkan distribusi frekuensi data numerik.

import matplotlib.pyplot as plt

# Data
data = np.random.randn(1000)

# Menambahkan judul dan label
plt.title("Histogram")
plt.xlabel("Nilai")
plt.ylabel("Frekuensi")

# Menampilkan plot
plt.show()

Penjelasan:

  • np.random.randn(1000): Menghasilkan 1000 data acak dari distribusi normal.
  • plt.hist(data, bins=30, color='purple', edgecolor='black'): Membuat histogram dengan 30 bin, warna ungu, dan tepi hitam.

# 5. Pie Chart

Pie chart digunakan untuk menampilkan proporsi kategori dalam bentuk lingkaran.

import matplotlib.pyplot as plt

# Data
labels = ['Python', 'C++', 'Ruby', 'Java']
sizes = [215, 130, 245, 210]
colors = ['gold', 'yellowgreen', 'lightcoral', 'lightskyblue']
explode = (0.1, 0, 0, 0)  # Memisahkan bagian pertama

# Membuat pie chart
plt.pie(sizes, explode=explode, labels=labels, colors=colors,
        autopct='%1.1f%%', shadow=True, startangle=140)

# Menambahkan judul
plt.title("Pie Chart: Bahasa Pemrograman Populer")

# Menampilkan plot
plt.axis('equal')  # Membuat pie chart berbentuk lingkaran
plt.show()

Penjelasan:

  • explode=(0.1, 0, 0, 0): Memisahkan bagian pertama dari pie chart.
  • autopct='%1.1f%%': Menampilkan persentase pada setiap bagian.
  • plt.axis('equal'): Memastikan pie chart berbentuk lingkaran.

Output:

# Membuat Subplots

Subplots memungkinkan pembuatan beberapa plot dalam satu figure.

import matplotlib.pyplot as plt

# Data
x = [0, 1, 2, 3, 4]
y1 = [0, 1, 4, 9, 16]
y2 = [0, 1, 2, 3, 4]

# Membuat subplots 1 baris dan 2 kolom
fig, axs = plt.subplots(1, 2, figsize=(10, 4))

# Plot pertama
axs[0].plot(x, y1, color='blue', marker='o')
axs[0].set_title("y = x²")
axs[0].set_xlabel("X")
axs[0].set_ylabel("Y")

# Plot kedua
axs[1].plot(x, y2, color='red', marker='x')
axs[1].set_title("y = x")
axs[1].set_xlabel("X")
axs[1].set_ylabel("Y")

# Menyesuaikan layout
plt.tight_layout()

# Menampilkan plot
plt.show()

Penjelasan:

  • plt.subplots(1, 2, figsize=(10, 4)): Membuat figure dengan 1 baris dan 2 kolom subplot, serta ukuran figure.
  • axs[0] dan axs[1]: Merujuk ke subplot pertama dan kedua.
  • plt.tight_layout(): Menyesuaikan layout agar tidak saling tumpang tindih.

Output:

# Advanced Plotting di Matplotlib

Matplotlib menyediakan berbagai opsi lanjutan untuk membuat visualisasi data yang lebih kompleks dan menarik.

# 1. Anotasi pada Plot

Anotasi membantu menyoroti atau menjelaskan poin-poin penting dalam plot.

import matplotlib.pyplot as plt

# Data
x = [0, 1, 2, 3, 4]
y = [0, 1, 4, 9, 16]

# Membuat plot garis
plt.plot(x, y, color='teal', marker='D')

# Menambahkan judul dan label
plt.title("Plot Garis dengan Anotasi")
plt.xlabel("Nilai X")
plt.ylabel("Nilai Y")

# Menambahkan anotasi
plt.annotate('Puncak', xy=(4, 16), xytext=(2, 10),
             arrowprops=dict(facecolor='black', shrink=0.05))

# Menampilkan plot
plt.show()

Penjelasan:

  • plt.annotate(): Menambahkan anotasi pada plot.
    • xy: Lokasi titik yang akan dianotasi.
    • xytext: Lokasi teks anotasi.
    • arrowprops: Mengatur tampilan panah yang menghubungkan teks ke titik.

Output:

# 2. Menggunakan Legends yang Lebih Kompleks

Legends membantu membedakan berbagai plot dalam satu figure.

import matplotlib.pyplot as plt

# Data
x = [0, 1, 2, 3, 4]
y1 = [0, 1, 4, 9, 16]
y2 = [0, 1, 2, 3, 4]
y3 = [0, -1, -2, -3, -4]

# Membuat plot garis
plt.plot(x, y1, label='y = x²', color='blue', linestyle='-')
plt.plot(x, y2, label='y = x', color='green', linestyle='--')
plt.plot(x, y3, label='y = -x', color='red', linestyle=':')

# Menambahkan judul dan label
plt.title("Plot Garis dengan Legends")
plt.xlabel("Nilai X")
plt.ylabel("Nilai Y")

# Menambahkan legenda di lokasi tertentu
plt.legend(loc='upper left')

# Menampilkan plot
plt.show()

Penjelasan:

  • label='...': Memberi label pada setiap plot garis.
  • plt.legend(loc='upper left'): Menampilkan legenda di sudut kiri atas plot.

Output:

# 3. Plot dengan Subplots dan Berbagai Jenis Plot

Menggabungkan berbagai jenis plot dalam satu figure menggunakan subplots.

import matplotlib.pyplot as plt

# Data
x = [0, 1, 2, 3, 4]
y_line = [0, 1, 4, 9, 16]
y_bar = [5, 7, 3, 8, 2]
y_scatter = [10, 15, 13, 17, 14]

# Membuat subplots 2x2
fig, axs = plt.subplots(2, 2, figsize=(10, 8))

# Plot garis
axs[0, 0].plot(x, y_line, color='blue', marker='o')
axs[0, 0].set_title("Plot Garis")
axs[0, 0].set_xlabel("X")
axs[0, 0].set_ylabel("Y")

# Bar chart
axs[0, 1].bar(x, y_bar, color='green')
axs[0, 1].set_title("Bar Chart")
axs[0, 1].set_xlabel("X")
axs[0, 1].set_ylabel("Nilai")

# Scatter plot
axs[1, 0].scatter(x, y_scatter, color='red', marker='^')
axs[1, 0].set_title("Scatter Plot")
axs[1, 0].set_xlabel("X")
axs[1, 0].set_ylabel("Y")

# Histogram
axs[1, 1].hist(y_scatter, bins=5, color='purple', edgecolor='black')
axs[1, 1].set_title("Histogram")
axs[1, 1].set_xlabel("Nilai")
axs[1, 1].set_ylabel("Frekuensi")

# Menyesuaikan layout
plt.tight_layout()

# Menampilkan plot
plt.show()

Penjelasan:

  • fig, axs = plt.subplots(2, 2, figsize=(10, 8)): Membuat figure dengan 2 baris dan 2 kolom subplot.
  • axs[row, col]: Merujuk ke subplot pada posisi tertentu.
  • Berbagai jenis plot digunakan dalam subplots yang berbeda untuk demonstrasi.

Output:

# 4. Mengatur Transparansi dan Warna pada Plot

Transparansi (alpha) dan warna dapat diatur untuk memperjelas tampilan plot, terutama saat plot saling tumpang tindih.

import matplotlib.pyplot as plt

# Data
x = [1, 2, 3, 4, 5]
y1 = [2, 3, 5, 7, 11]
y2 = [1, 4, 6, 8, 10]

# Membuat scatter plot dengan transparansi
plt.scatter(x, y1, color='blue', alpha=0.5, label='Dataset 1')
plt.scatter(x, y2, color='red', alpha=0.5, label='Dataset 2')

# Menambahkan judul dan label
plt.title("Scatter Plot dengan Transparansi")
plt.xlabel("X")
plt.ylabel("Y")

# Menambahkan legenda
plt.legend()

# Menampilkan plot
plt.show()

Penjelasan:

  • alpha=0.5: Mengatur transparansi plot menjadi 50%, sehingga plot yang tumpang tindih dapat terlihat.
  • label='...': Memberi label untuk legenda.

Output:

# 5. Menambahkan Grid dan Memodifikasi Ticks

Grid dan ticks dapat membantu dalam membaca plot dengan lebih baik.

import matplotlib.pyplot as plt

# Data
x = [0, 1, 2, 3, 4, 5]
y = [0, 1, 4, 9, 16, 25]

# Membuat plot garis
plt.plot(x, y, color='cyan', marker='D')

# Menambahkan judul dan label
plt.title("Plot dengan Grid dan Custom Ticks")
plt.xlabel("Nilai X")
plt.ylabel("Nilai Y")

# Menambahkan grid
plt.grid(True, which='both', linestyle='--', linewidth=0.5)

# Mengatur ticks
plt.xticks(range(0, 6, 1))
plt.yticks(range(0, 26, 5))

# Menampilkan plot
plt.show()

Penjelasan:

  • plt.grid(True, which='both', linestyle='--', linewidth=0.5): Menambahkan grid pada kedua sumbu dengan garis putus-putus dan ketebalan 0.5.
  • plt.xticks(range(0, 6, 1)): Mengatur ticks sumbu X setiap 1 unit.
  • plt.yticks(range(0, 26, 5)): Mengatur ticks sumbu Y setiap 5 unit.

# Integrasi Matplotlib dengan Pandas

Matplotlib dapat digunakan secara langsung dengan Pandas untuk membuat visualisasi data dari DataFrame.

# Contoh: Plot dari DataFrame Pandas 

import pandas as pd
import matplotlib.pyplot as plt

# Membuat DataFrame
data = {
    'Bulan': ['Jan', 'Feb', 'Mar', 'Apr', 'Mei'],
    'Penjualan': [250, 300, 400, 350, 500]
}
df = pd.DataFrame(data)

# Membuat plot garis dari DataFrame
df.plot(x='Bulan', y='Penjualan', kind='line', marker='o', color='magenta')

# Menambahkan judul dan label
plt.title("Penjualan Bulanan")
plt.xlabel("Bulan")
plt.ylabel("Jumlah Penjualan")

# Menampilkan plot
plt.show()

Penjelasan:

  • df.plot(...): Menggunakan metode plot dari Pandas untuk membuat plot langsung dari DataFrame.
  • Parameter x, y, kind, marker, dan color digunakan untuk mengatur jenis plot dan tampilannya.

Output:

# Contoh: Bar Chart dari DataFrame Pandas 

import pandas as pd
import matplotlib.pyplot as plt

# Membuat DataFrame
data = {
    'Produk': ['A', 'B', 'C', 'D'],
    'Terjual': [150, 200, 300, 250]
}
df = pd.DataFrame(data)

# Membuat bar chart dari DataFrame
df.plot(x='Produk', y='Terjual', kind='bar', color='skyblue', legend=False)

# Menambahkan judul dan label
plt.title("Jumlah Produk Terjual")
plt.xlabel("Produk")
plt.ylabel("Jumlah Terjual")

# Menambahkan anotasi jumlah terjual di atas setiap bar
for index, value in enumerate(df['Terjual']):
    plt.text(index, value + 5, str(value), ha='center')

# Menampilkan plot
plt.show()

Penjelasan:

  • kind='bar': Mengatur jenis plot menjadi bar chart.
  • legend=False: Menonaktifkan legenda.
  • Loop for digunakan untuk menambahkan anotasi teks di atas setiap bar.

Output:

# 3. Menambahkan Subplot dengan Grid

Menggunakan grid untuk menambahkan beberapa subplot dalam satu figure dengan pengaturan yang rapi.

import matplotlib.pyplot as plt
import numpy as np

# Data
x = np.linspace(0, 10, 100)
y1 = np.sin(x)
y2 = np.cos(x)

# Membuat subplots 2x1
fig, axs = plt.subplots(2, 1, figsize=(10, 8))

# Plot pertama: Sinus
axs[0].plot(x, y1, color='blue')
axs[0].set_title("Fungsi Sinus")
axs[0].set_xlabel("X")
axs[0].set_ylabel("sin(X)")
axs[0].grid(True)

# Plot kedua: Cosinus
axs[1].plot(x, y2, color='red')
axs[1].set_title("Fungsi Cosinus")
axs[1].set_xlabel("X")
axs[1].set_ylabel("cos(X)")
axs[1].grid(True)

# Menyesuaikan layout
plt.tight_layout()

# Menampilkan plot
plt.show()

Penjelasan:

  • np.linspace(0, 10, 100): Membuat array x dengan 100 titik antara 0 hingga 10.
  • fig, axs = plt.subplots(2, 1, figsize=(10, 8)): Membuat figure dengan 2 baris dan 1 kolom subplot.
  • axs[0].grid(True): Menambahkan grid pada subplot pertama.
  • plt.tight_layout(): Menyesuaikan layout agar subplot tidak saling tumpang tindih.

Output:
