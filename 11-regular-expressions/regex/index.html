# Regular Expressions

Selamat datang! Regex adalah alat yang sangat berguna untuk memanipulasi dan menganalisis teks. Bagi mahasiswa matematika, kemampuan ini dapat membantu dalam pengolahan data, pembuatan laporan, automasi tugas-tugas sehari-hari, serta analisis teks yang kompleks.

# Apa itu Regex?

Regex adalah singkatan dari Regular Expressions, yaitu sekumpulan karakter yang membentuk pola pencarian. Pola ini digunakan untuk mencocokkan, mencari, atau mengganti teks dalam string (kalimat). Regex memungkinkan kita untuk melakukan operasi pencarian dan manipulasi teks dengan efisien dan fleksibel.

# Mengapa Regex Penting?

  • Bisa memproses teks dalam jumlah besar dengan cepat.
  • Dapat digunakan untuk berbagai kebutuhan, mulai dari validasi hingga ekstraksi data.
  • Banyak dipakai dalam berbagai bahasa pemrograman, termasuk Python.
  • Membantu dalam membersihkan dan menyiapkan data untuk analisis lebih lanjut.

# Dasar-dasar Regex

# Karakter Literal

Karakter yang dicocokkan secara langsung. Misalnya, pola a akan mencari huruf "a" dalam teks.

a

Contoh:

  • cat akan mencocokkan kata "cat" dalam "The cat sat on the mat."
  • 123 akan mencocokkan "123" dalam "Order number: 12345."

# Metakarakter

Karakter khusus yang memiliki arti tertentu dalam regex. Beberapa metakarakter yang sering digunakan:

  • . : Mencocokkan sembarang karakter kecuali newline.
  • ^ : Menandai awal baris.
  • $ : Menandai akhir baris.
  • \d : Mencocokkan digit (angka 0-9).
  • \w : Mencocokkan karakter kata (huruf, angka, dan underscore).
  • \s : Mencocokkan spasi putih (space, tab, newline).
  • \b : Mencocokkan batas kata.
  • \B : Mencocokkan bukan batas kata.
  • \D, \W, \S : Kebalikan dari \d, \w, \s.

Contoh:

  • \d+ akan mencocokkan satu atau lebih digit, seperti "123".
  • \w{3} akan mencocokkan tepat tiga karakter kata, seperti "abc".

# Kuantifier

Menentukan jumlah kemunculan karakter atau grup yang ingin dicocokkan.

  • * : 0 atau lebih kali.
  • + : 1 atau lebih kali.
  • ? : 0 atau 1 kali.
  • {n} : Tepat n kali.
  • {n,} : Minimal n kali.
  • {n,m} : Antara n dan m kali.

Contoh:

  • a* akan mencocokkan "", "a", "aa", "aaa", dll.
  • a{2,4} akan mencocokkan "aa", "aaa", atau "aaaa".

# Kelas Karakter

Menentukan sekumpulan karakter yang bisa dicocokkan di posisi tertentu.

  • [abc] : Mencocokkan salah satu dari a, b, atau c.
  • [a-z] : Mencocokkan huruf kecil dari a sampai z.
  • [A-Z] : Mencocokkan huruf besar dari A sampai Z.
  • [0-9] : Mencocokkan digit dari 0 sampai 9.
  • [^a-z] : Mencocokkan karakter yang bukan huruf kecil dari a sampai z.
  • [a-zA-Z0-9_] : Mencocokkan karakter alfanumerik dan underscore.

Contoh:

  • [0-9]+ akan mencocokkan satu atau lebih digit, seperti "123".
  • [A-Za-z]+ akan mencocokkan satu atau lebih huruf, baik besar maupun kecil.

# Grup dan Capturing

Mengelompokkan bagian dari pola regex untuk diterapkan kuantifier atau untuk di-referensi kembali.

  • () : Membuat grup.
  • (?: ) : Membuat grup non-capturing.
  • (?P<name> ) : Membuat grup dengan nama.

Contoh:

  • (abc)+ akan mencocokkan "abc", "abcabc", "abcabcabc", dll.
  • (?P<digit>\d+) akan mencocokkan satu atau lebih digit dan memberi nama grup sebagai "digit".

# Alternation

Menggunakan simbol | untuk mencocokkan salah satu dari beberapa pola.

Contoh:

  • cat|dog akan mencocokkan "cat" atau "dog".
  • Mr\.|Mrs\.|Ms\. akan mencocokkan "Mr.", "Mrs.", atau "Ms.".

# Lookahead dan Lookbehind

Memungkinkan pencocokan berbasis konteks tanpa termasuk karakter tersebut dalam hasil.

  • Lookahead Positif: (?=...)
  • Lookahead Negatif: (?!...)
  • Lookbehind Positif: (?<=...)
  • Lookbehind Negatif: (?<!...)

Contoh:

  • \d+(?= dollars) akan mencocokkan angka yang diikuti oleh " dollars" tanpa memasukkan " dollars" dalam hasil.
  • (?<!\$)\d+ akan mencocokkan angka yang tidak diawali dengan tanda dolar.

# Regex di Python

Python menyediakan modul re yang memungkinkan penggunaan regex dalam program. Modul ini menawarkan berbagai fungsi untuk pencocokan, pencarian, dan manipulasi teks menggunakan regex.

# Import Modul re

Sebelum menggunakan regex di Python, kita perlu mengimpor modul re.

import re

# Fungsi-fungsi Dasar

  1. re.match()

    Mencocokkan pola di awal string. Jika cocok, mengembalikan objek match; jika tidak, mengembalikan None.

    result = re.match(r'\d+', '123abc')
if result:
    print(result.group())  # Output: 123

  2. re.search()

    Mencari pola di seluruh string. Mengembalikan objek match pertama yang ditemukan atau None jika tidak ada.

    result = re.search(r'\d+', 'abc123')
if result:
    print(result.group())  # Output: 123

  3. re.findall()

    Mengembalikan semua kemunculan pola dalam bentuk list.

    results = re.findall(r'\d+', 'abc123def456')
print(results)  # Output: ['123', '456']

  4. re.finditer()

    Mengembalikan iterator yang menghasilkan objek match untuk setiap kemunculan pola.

    for match in re.finditer(r'\d+', 'abc123def456'):
    print(match.group())
# Output:
# 123
# 456

  5. re.sub()

    Mengganti pola yang ditemukan dengan string lain.

    new_text = re.sub(r'\d+', '#', 'abc123def456')
print(new_text)  # Output: abc#def#

  6. re.split()

    Membagi string berdasarkan pola regex.

    parts = re.split(r'\W+', 'Hello, world! 123')
print(parts)  # Output: ['Hello', 'world', '123']

# Flags (Penanda)

Modul re menyediakan berbagai flag untuk mengubah perilaku pencocokan.

  • re.IGNORECASE atau re.I : Mengabaikan huruf besar/kecil.
  • re.MULTILINE atau re.M : ^ dan $ mencocokkan awal dan akhir setiap baris.
  • re.DOTALL atau re.S : . mencocokkan newline juga.
  • re.VERBOSE atau re.X : Memungkinkan regex ditulis dengan spasi dan komentar untuk keterbacaan.
  • re.UNICODE atau re.U : Menggunakan karakter Unicode.

Contoh:

pattern = re.compile(r'\d+', re.IGNORECASE | re.MULTILINE)

# Grup dan Named Groups

Menggunakan grup untuk menangkap bagian dari pola yang dicocokkan, yang kemudian dapat diakses secara terpisah.

Contoh:

import re

teks = "Nama: John Doe, Usia: 20"
pola = r'Nama: (?P<nama>[A-Za-z ]+), Usia: (?P<usia>\d+)'

match = re.search(pola, teks)
if match:
    print(match.group('nama'))  # Output: John Doe
    print(match.group('usia'))  # Output: 20

# Escaping Karakter Khusus

Jika perlu mencocokkan karakter yang memiliki arti khusus dalam regex, gunakan backslash \ untuk meng-escape-nya.

Contoh:

  • Untuk mencocokkan titik (.), gunakan \..
  • Untuk mencocokkan tanda kurung ( atau ), gunakan \( atau \).
pola = r'\d+\.\d+'  # Mencocokkan angka desimal seperti 3.14

# Contoh Penggunaan dalam Matematika

# Ekstraksi Angka dari Ekspresi Matematika

Misalnya, kalian memiliki data berupa string yang berisi berbagai ekspresi matematika, dan ingin mengekstrak semua angka yang muncul.

import re

teks = "Persamaan: 2x + 3 = 7. Solusi x = 2"

angka = re.findall(r'\d+', teks)
print(angka)  # Output: ['2', '3', '7', '2']

# Validasi Ekspresi Matematika

Memastikan bahwa sebuah string adalah ekspresi matematika yang valid sesuai pola tertentu.

import re

pola = r'^[\d\w\+\-\*\/\^\(\)\s]+$'

ekspresi1 = "3x^2 + 2x - 5"
ekspresi2 = "3x^2 + 2x - five"

print(bool(re.match(pola, ekspresi1)))  # Output: True
print(bool(re.match(pola, ekspresi2)))  # Output: False

# Parsing Data Numerik dari Teks Matematika

Mengambil data numerik dari teks untuk analisis lebih lanjut, seperti menghitung statistik atau membuat grafik.

import re

teks = """
Data Observasi:
- Tahun 2020: 150 unit
- Tahun 2021: 175 unit
- Tahun 2022: 200 unit
"""

tahun = re.findall(r'Tahun (\d{4})', teks)
unit = re.findall(r'(\d+) unit', teks)

print(tahun)  # Output: ['2020', '2021', '2022']
print(unit)   # Output: ['150', '175', '200']

# Studi Kasus: Ekstraksi Data Matematika

Misalnya ada data hasil survei yang berisi tanggapan dalam format teks. Lalu kita mengekstrak semua nilai numerik untuk analisis lebih lanjut.

import re

data_survei = """
Respondent 1: Usia 21, Skor 85
Respondent 2: Usia 22, Skor 90
Respondent 3: Usia 20, Skor 78
Respondent 4: Usia 23, Skor 88
"""

# Ekstrak semua usia
usia = re.findall(r'Usia (\d+)', data_survei)
print(usia)  # Output: ['21', '22', '20', '23']

# Ekstrak semua skor
skor = re.findall(r'Skor (\d+)', data_survei)
print(skor)  # Output: ['85', '90', '78', '88']

# Konversi ke integer untuk analisis
usia = list(map(int, usia))
skor = list(map(int, skor))

# Hitung rata-rata usia dan skor
rata_usia = sum(usia) / len(usia)
rata_skor = sum(skor) / len(skor)

print(f"Rata-rata Usia: {rata_usia}")  # Output: Rata-rata Usia: 21.5
print(f"Rata-rata Skor: {rata_skor}")  # Output: Rata-rata Skor: 85.25

# Daftar Istilah

  • String: Urutan karakter yang diapit oleh tanda kutip.
  • Pattern: Pola yang digunakan untuk mencocokkan string dalam regex.
  • Match: Proses mencocokkan pola regex dengan string.
  • Group: Bagian dari pola regex yang diapit oleh tanda kurung ().
  • Flag: Opsi tambahan yang mengubah perilaku pencocokan regex.
  • Quantifier: Simbol yang menentukan jumlah kemunculan karakter atau grup.
  • Backreference: Referensi ke grup yang sudah didefinisikan sebelumnya dalam pola regex.
