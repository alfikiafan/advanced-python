# Inheritance

# Apa Itu Inheritance?

Bayangkan kamu punya keluarga besar. Sifat dan kebiasaan tertentu dari orang tua bisa diwariskan ke anak. Dalam dunia kode, kita punya konsep inheritance atau pewarisan, yang artinya kelas child (anak) bisa mewarisi properti atau metode dari kelas parent (orang tua), sehingga kita tidak perlu menulis ulang hal yang sama.

# Contoh Kasus Sederhana:

Kita akan mulai dengan membuat kelas Person, yaitu kelas parent yang punya dua atribut: name dan id. Kemudian kita tambahkan metode untuk menampilkan nama dan ID.

class Person:
    def __init__(self, name, id):
        self.name = name  # Menyimpan nama di atribut 'name'
        self.id = id      # Menyimpan ID di atribut 'id'
    
    def display(self):
        print(f"Nama: {self.name}, ID: {self.id}")  # Metode untuk menampilkan nama dan ID

Penjelasan Kode:

  • __init__: Merupakan constructor yang dipanggil saat objek dibuat. Ini berfungsi untuk menginisialisasi atribut seperti name dan id.
  • display(): Metode yang menampilkan nilai dari name dan id dalam format kalimat.

Sekarang, kita buat objek Person:

# Membuat objek dari kelas Person
person = Person("Siti", 123)
person.display()  # Output: Nama: Siti, ID: 123

Penjelasan:
Di sini kita membuat objek person dengan nama Siti dan ID 123, kemudian memanggil metode display() untuk menampilkan informasi person tersebut.

Output:

Nama: Siti, ID: 123

# Membuat Kelas Anak yang Mewarisi dari Kelas Parent

Sekarang kita buat kelas anak, yaitu Employee, yang mewarisi dari Person. Kelas ini akan memiliki semua atribut dan metode yang dimiliki oleh Person.

class Employee(Person):  # Employee mewarisi dari Person
    def print_employee(self):
        print("Ini adalah kelas Employee")  # Metode tambahan khusus untuk kelas Employee

Penjelasan Kode:

  • class Employee(Person):: Menandakan bahwa Employee adalah kelas anak yang mewarisi dari kelas Person.
  • print_employee(): Metode tambahan khusus untuk kelas Employee.

Sekarang kita buat objek dari kelas Employee dan lihat bagaimana ia mewarisi metode dari Person.

# Membuat objek dari kelas Employee
pegawai = Employee("Ayu", 456)
pegawai.display()          # Memanggil metode dari kelas _parent_ (Person)
pegawai.print_employee()  # Memanggil metode dari kelas anak

Penjelasan:

  • pegawai.display(): Memanggil metode display() dari kelas Person, menunjukkan bahwa Employee mewarisi semua metode Person.
  • pegawai.print_employee(): Memanggil metode khusus yang hanya ada di Employee.

Output:

Nama: Ayu, ID: 456
Ini adalah kelas Employee

# Menggunakan super() untuk Memanggil Konstruktor Kelas Parent

Jika kamu ingin menambahkan lebih banyak properti ke kelas anak, tapi tetap ingin menggunakan properti dari kelas parent, kamu bisa menggunakan fungsi super() untuk memanggil konstruktor orang tua.

Contohnya, mari kita tambahkan atribut salary (gaji) dan position (jabatan) ke kelas Employee, tapi tetap ingin name dan id diwarisi dari kelas Person.

class Person:
    def __init__(self, name, idnumber):
        self.name = name               # Atribut 'name' dari kelas Person
        self.idnumber = idnumber       # Atribut 'idnumber' dari kelas Person

class Employee(Person):
    def __init__(self, name, idnumber, salary, position):
        # Panggil constructor dari kelas _parent_ untuk menginisialisasi 'name' dan 'idnumber'
        super().__init__(name, idnumber)
        self.salary = salary           # Atribut tambahan 'salary' untuk kelas Employee
        self.position = position       # Atribut tambahan 'position' untuk kelas Employee

# Membuat objek Employee
pegawai = Employee("Rini", 789, 5000000, "Manager")
print(f"{pegawai.name} memiliki ID {pegawai.idnumber}, gaji sebesar Rp{pegawai.salary}, dan menjabat sebagai {pegawai.position}.")

Penjelasan Kode:

  • super().__init__(name, idnumber): Memanggil konstruktor dari kelas parent, yaitu Person, untuk menginisialisasi atribut name dan idnumber.
  • self.salary dan self.position: Atribut tambahan yang ditambahkan khusus untuk kelas Employee.
  • print(): Menampilkan informasi lengkap tentang objek Employee dalam bentuk kalimat.

Output:

Rini memiliki ID 789, gaji sebesar Rp5000000, dan menjabat sebagai Manager.

# Jenis-jenis Inheritance di Python

Inheritance di Python juga memiliki berbagai bentuk, mirip dengan hubungan keluarga. Mari kita bahas satu per satu dengan contoh.

# 1. Single Inheritance: Anak mewarisi dari satu orang tua saja. 

class Person:
    def __init__(self, name):
        self.name = name

class Employee(Person):  # Employee hanya mewarisi dari satu kelas, yaitu Person
    def __init__(self, name, department):
        super().__init__(name)
        self.department = department

    def display_employee(self):
        print(f"{self.name} bekerja di departemen {self.department}.")
        
# Membuat objek dari kelas Employee
pegawai = Employee("Dedi", "IT")
pegawai.display_employee()  # Output: Dedi bekerja di departemen IT.

Penjelasan Kode:

  • Employee mewarisi dari satu kelas, yaitu Person.
  • Employee menambahkan atribut department dan metode display_employee().

Output:

Dedi bekerja di departemen IT.

# 2. Multiple Inheritance: Anak bisa mewarisi dari lebih dari satu orang tua. 

class Base1:
    def __init__(self):
        self.str1 = "Text dari Base1"
        print("Base1 diinisialisasi")

class Base2:
    def __init__(self):
        self.str2 = "Text dari Base2"
        print("Base2 diinisialisasi")

class Derived(Base1, Base2):  # Mewarisi dari Base1 dan Base2
    def __init__(self):
        Base1.__init__(self)  # Memanggil constructor Base1
        Base2.__init__(self)  # Memanggil constructor Base2
        print("Derived diinisialisasi")
    
    def printStrs(self):
        print(f"{self.str1} dan {self.str2}")

# Membuat objek dari kelas Derived
ob = Derived()
ob.printStrs()  # Output: Text dari Base1 dan Text dari Base2

Penjelasan Kode:

  • Derived mewarisi dari dua kelas, Base1 dan Base2.
  • Konstruktor Derived memanggil konstruktor dari kedua kelas induk.
  • Metode printStrs() menampilkan atribut dari kedua kelas induk.

Output:

Base1 diinisialisasi
Base2 diinisialisasi
Derived diinisialisasi
Text dari Base1 dan Text dari Base2

# 3. Multilevel Inheritance: Seperti hubungan kakek, ayah, dan anak, dimana anak mewarisi dari orang tua, dan orang tua mewarisi dari kakek. 

class Grandparent:
    def __init__(self, name):
        self.name = name

class Parent(Grandparent):
    def __init__(self, name, age):
        super().__init__(name)
        self.age = age

class Child(Parent):
    def __init__(self, name, age, address):
        super().__init__(name, age)
        self.address = address

    def display_info(self):
        print(f"{self.name} berusia {self.age} tahun dan tinggal di {self.address}.")

# Membuat objek dari kelas Child
anak = Child("Budi", 25, "Jakarta")
anak.display_info()  # Output: Budi berusia 25 tahun dan tinggal di Jakarta.

Penjelasan Kode:

  • Child mewarisi dari Parent, yang juga mewarisi dari Grandparent.
  • Dengan menggunakan super(), setiap kelas memanggil konstruktor dari kelas induknya untuk menginisialisasi atribut yang diwariskan.
  • Metode display_info() menampilkan semua atribut yang dimiliki oleh Child dalam bentuk kalimat.

Output:

Budi berusia 25 tahun dan tinggal di Jakarta.

# 4. Hierarchical Inheritance: Satu kelas orang tua bisa punya beberapa anak. 

class Parent:
    def __init__(self, name):
        self.name = name

class Child1(Parent):
    def __init__(self, name, hobby):
        super().__init__(name)
        self.hobby = hobby

    def display_child1(self):
        print(f"{self.name} memiliki hobi {self.hobby}.")

class Child2(Parent):
    def __init__(self, name, favorite_subject):
        super().__init__(name)
        self.favorite_subject = favorite_subject

    def display_child2(self):
        print(f"Mata pelajaran favorit {self.name} adalah {self.favorite_subject}.")

# Membuat objek dari kelas Child1 dan Child2
anak1 = Child1("Dina", "Membaca")
anak2 = Child2("Eka", "Matematika")

anak1.display_child1()  # Output: Dina memiliki hobi Membaca.
anak2.display_child2()  # Output: Mata pelajaran favorit Eka adalah Matematika.

Penjelasan Kode:

  • Child1 dan Child2 keduanya mewarisi dari kelas Parent.
  • Masing-masing kelas anak menambahkan atribut dan metode spesifik mereka sendiri.
  • Objek anak1 dan anak2 memanfaatkan atribut dan metode dari kelas induk Parent serta atribut dan metode mereka sendiri.

Output:

Dina memiliki hobi Membaca.
Mata pelajaran favorit Eka adalah Matematika.

# 5. Hybrid Inheritance: Kombinasi dari berbagai jenis inheritance di atas. 

class Parent1:
    def __init__(self, name):
        self.name = name

class Parent2:
    def __init__(self, hobby):
        self.hobby = hobby

class Child1(Parent1):
    def __init__(self, name, favorite_color):
        super().__init__(name)
        self.favorite_color = favorite_color

    def display_child1(self):
        print(f"Warna favorit {self.name} adalah {self.favorite_color}.")

class Child2(Parent1, Parent2):
    def __init__(self, name, hobby, age):
        Parent1.__init__(self, name)
        Parent2.__init__(self, hobby)
        self.age = age

    def display_child2(self):
        print(f"{self.name} berusia {self.age} tahun dan memiliki hobi {self.hobby}.")

# Membuat objek dari kelas Child1 dan Child2
anak1 = Child1("Feri", "Biru")
anak2 = Child2("Gina", "Memasak", 30)

anak1.display_child1()  # Output: Warna favorit Feri adalah Biru.
anak2.display_child2()  # Output: Gina berusia 30 tahun dan memiliki hobi Memasak.

Penjelasan Kode:

  • Child1 mewarisi dari Parent1.
  • Child2 mewarisi dari Parent1 dan Parent2 (multiple inheritance).
  • Ini adalah contoh Hybrid Inheritance yang menggabungkan single dan multiple inheritance.
  • Setiap kelas anak menambahkan atribut dan metode spesifik mereka sendiri.

Output:

Warna favorit Feri adalah Biru.
Gina berusia 30 tahun dan menikmati hobi Memasak.

# Contoh Penggunaan Inheritance

Kamu bisa juga menambahkan properti khusus di kelas anak. Misalnya, kita punya kelas Person dan kita ingin menambahkan tanggal lahir di kelas anak:

class Person:
    def __init__(self, name, age):
        self.name = name       # Atribut 'name' dari kelas Person
        self.age = age         # Atribut 'age' dari kelas Person

    def display(self):
        print(f"Nama: {self.name}, Umur: {self.age} tahun.")  # Metode untuk menampilkan nama dan umur

class Student(Person):
    def __init__(self, name, age, dob):
        super().__init__(name, age)  # Memanggil constructor kelas parent
        self.dob = dob               # Atribut tambahan 'dob' untuk kelas Student

    def display_info(self):
        print(f"Nama: {self.name}, Umur: {self.age} tahun, Tanggal Lahir: {self.dob}.")  # Metode untuk menampilkan informasi lengkap

# Membuat objek Student
murid = Student("Dion", 20, "16-03-2004")
murid.display()        # Memanggil metode dari kelas parent
murid.display_info()   # Memanggil metode dari kelas anak

Penjelasan Kode:

  • Student mewarisi dari Person dan menambahkan atribut dob (tanggal lahir).
  • Menggunakan super().__init__(name, age) untuk menginisialisasi atribut dari kelas Person.
  • Metode display_info() menampilkan semua atribut, termasuk yang baru ditambahkan, dalam bentuk kalimat.

Output:

Nama: Dion, Umur: 20 tahun.
Nama: Dion, Umur: 20 tahun, Tanggal Lahir: 16-03-2004.
